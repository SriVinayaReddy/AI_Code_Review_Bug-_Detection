# 🤖 Generative AI for Predictive Code Review & Bug Detection

## 📌 Overview
This project is a Generative AI-based system that performs automated code review and bug detection. It combines static analysis tools with Large Language Models (LLMs) to identify errors, suggest improvements, and generate corrected code.

---

## 🎯 Features

- 🔍 Detects syntax errors and bugs
- 🧠 AI-powered code review using LLM
- 💡 Suggests improvements and best practices
- 🛠 Generates corrected code
- 📊 Static analysis using Pylint
- 🌐 Simple UI using Streamlit

---

## 🏗️ Project Architecture

User Input Code  
⬇  
Static Analysis (Pylint)  
⬇  
Generative AI Model (LLM)  
⬇  
Bug Detection + Suggestions  
⬇  
Improved Code Output  

---

## ⚙️ Tech Stack

- Python  
- Streamlit  
- OpenAI API (LLM)  
- Pylint
